# Vibe Coding Practice

바닐라 HTML/CSS/JS로 만드는 프론트엔드 연습 프로젝트입니다.  
빌드 도구나 프레임워크 없이 브라우저에서 바로 실행됩니다.

## 연습 목록

| # | 이름 | 설명 |
|---|------|------|
| 00 | Portfolio | 전체 프로젝트 인덱스 페이지 |
| 01 | Hello World | 기본 페이지 구성 연습 |
| 02 | Profile | 프로필 카드 UI |
| 03 | Counter | 클릭 카운터 |
| 04 | To-do | 할 일 목록 앱 (로컬스토리지 저장) |
| 05 | Exchange | 환율 변환기 |
| 06 | Snake | 스네이크 게임 (난이도·레벨·모바일 지원) |

## 실행 방법

별도 설치 없이 `.html` 파일을 브라우저로 열면 바로 실행됩니다.

## 파일 구조

```
NN.exerciseName.html   ← 마크업 + 스크립트
NN.exerciseName.css    ← 스타일
```

## 기술 스택

- Vanilla HTML / CSS / JavaScript
- No frameworks, no dependencies
